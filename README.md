肥波的淤青 (Feibo's Bruise) - 轻量级社交 Web 应用
“肥波的淤青”是一个基于 Python Flask 框架开发的轻量级、响应式社交 Web 应用。它集成了即时通讯、灵魂匹配、好友系统、动态空间以及完整的后台管理系统，致力于提供一个纯粹、有趣的在线社交体验。

✨ 核心功能 (Features)
💬 纯粹的聊天体验

大厅群聊：所有在线用户可以在大厅畅所欲言。

私密单聊：支持发送文本、图片、文件以及实时语音录制。

状态同步：支持未读消息红点角标提示，以及高级的“消息软删除”实时同步功能（删除后全网无痕消失）。

💞 灵魂在线匹配

根据“省份”、“性别”以及“匹配偏好（同城/异性/同性等）”进行在线雷达扫描。

匹配成功后自动发送破冰消息并跳转至私聊界面。

👥 闭环的好友系统

支持发送好友请求、同意请求。

智能防骚扰：私聊超过 5 句话且非好友关系时，系统会自动弹出加好友的唯美提示。

🌟 动态空间 (朋友圈)

发布图文动态，支持点赞、取消点赞。

支持在动态下方展开评论区进行实时评论。

专属收藏夹：一键收藏喜欢的动态，可在“我的”页面随时回顾。

动态主人及管理员拥有物理删除动态的权限。

🛡️ 本地专属管理后台

独立的 /admin 路由，强制仅限 127.0.0.1 (Localhost) 访问，拒绝外网刺探。

实时查看在线人数、注册人数、动态总数。

一键任命/撤销超级管理员（Admin）。

Admin 用户在前台拥有专属红底徽章，并拥有删除全站任何消息和动态的最高权限。

🔐 邀请码注册制

采用严格的“邀请码+哈希密码”注册机制，防止恶意批量注册，确保社区质量。

🛠️ 技术栈 (Tech Stack)
后端 (Backend): Python, Flask, Flask-SQLAlchemy, Werkzeug

数据库 (Database): SQLite (轻量、无需配置、开箱即用)

前端 (Frontend): HTML5, CSS3, JavaScript (jQuery 链式异步轮询)

UI 框架: Bootstrap 5, Bootstrap Icons

🚀 快速开始 (Quick Start)
1. 环境准备
请确保你的电脑上已安装 Python 3.8 或更高版本。

2. 安装依赖
在项目根目录下打开终端，运行以下命令安装必需的 Python 库：

Bash
pip install flask flask-sqlalchemy werkzeug
3. 运行应用与初始化数据库
首次运行应用时，SQLAlchemy 会自动在根目录下创建 site.db 数据库文件及所有表结构。

Bash
python app.py
看到 * Running on http://127.0.0.1:5000 后，先不要急着注册，请保持终端运行，另开一个终端进行下一步。

4. 生成邀请码 (必须)
由于系统开启了邀请码注册限制，你需要先生成邀请码才能注册账号。运行项目内的生成脚本：

Bash
python generate_codes.py
控制台会打印出 5 个类似 A8KF92XP 的邀请码，复制其中一个。

5. 访问应用
打开浏览器，访问：

前台入口: http://127.0.0.1:5000/ (使用刚才的邀请码注册并登录，第一个注册的用户将自动成为超级管理员)

后台管理: http://127.0.0.1:5000/admin (只能在运行代码的本机上访问)

💡 开发与部署建议 (Deployment Tips)
测试匹配功能：匹配功能需要至少两人同时参与。建议在电脑上使用正常的浏览器登录账号 A，再使用**浏览器的“无痕/隐身模式”**登录账号 B 进行测试。

关于卡顿优化：本项目前端采用了安全的链式异步轮询 (setTimeout Chaining) 获取新消息，完美解决了传统 setInterval 在网络波动时造成的请求堆积和页面假死问题。

上云部署：如果要将项目部署到公网服务器（如阿里云、腾讯云），请将运行参数修改为 app.run(host='0.0.0.0', port=80)，并确保服务器的安全组开放了对应端口。部署后若需访问后台，可通过 SSH 隧道映射本地端口以绕过 local_only 限制。
